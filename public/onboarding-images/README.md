# Onboarding Images

Place the following images in this directory:

## Default Background

## Text-to-Speech Tour

## Related Documents Tour
6. **related-docs-empty.png** - Screenshot of the empty Related Documents panel showing "Open a document to see related documents"
7. **related-docs-with-document.png** - Screenshot showing Related Documents panel with a document card and "AI is building the relationship..." message
8. **relevance-analysis.png** - Screenshot of the Relevance Analysis panel showing 62% relevance with breakdown by keywords, topics, themes, and summary
9. **document-relationship-graph.png** - Screenshot of the Document Relationship Graph modal showing nodes and edges with relevance legend

The images will automatically have a blur filter applied to protect text privacy (except the default background which uses an overlay).

## Image Requirements

- Format: PNG
- Recommended size: 800-1200px width (default background should match viewport size)
- The images will be automatically blurred (4px blur filter) when displayed in onboarding tooltips
