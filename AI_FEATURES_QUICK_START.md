# AI Chat & Notes - Quick Start Guide

## ğŸš€ Quick Start (60 seconds)

### 1. Open AI Chat
- Click the chat icon in the header OR
- Use keyboard shortcut (if configured)

### 2. Get AI Clarification
1. Select confusing text in your document
2. **Right-click** on the selection
3. Choose **"Ask AI for Clarification"**
4. Read the simplified explanation in the chat panel
5. (Optional) Click "Save Note" to keep it

### 3. Get Further Reading Suggestions
1. Select interesting text you want to explore
2. **Right-click** on the selection
3. Choose **"Get Further Reading"**
4. Review suggested topics and resources
5. (Optional) Save as a note for later

### 4. Save Quick Notes
1. Select any text in your document
2. **Right-click** on the selection
3. Choose **"Save as Note"**
4. Access later in Notes Panel

## ğŸ’¡ Pro Tips

### Chat Panel Controls
- **Expand/Collapse**: Click the maximize/minimize icon (top right)
- **Minimize**: Click the chevron to collapse to header bar
- **Close**: Click X or click backdrop outside panel

### Notes Context Toggle
- **Blue File Icon** = Notes context ENABLED (AI knows your notes)
- **Gray File Icon** = Notes context DISABLED (only document)
- Toggle by clicking the file icon in chat header

### Best Practices
1. **Select meaningful chunks** - Include complete sentences for better context
2. **Enable notes context** - Let AI learn from your previous insights
3. **Save useful responses** - Hover over AI messages and click "Save Note"
4. **Experiment with both modes** - Clarification vs Further Reading serve different purposes

## ğŸ”§ Troubleshooting

### Context menu not appearing?
- Make sure you have text selected
- Try selecting a bit more text (at least a few words)
- Check that you're right-clicking on the selection

### AI not responding?
- Check internet connection
- Verify API keys are configured in `.env.local`
- Try the request again (may be temporary API issue)

### Notes not saving?
- Ensure you have a document loaded
- Check browser console for errors
- Try refreshing the page

### Chat panel too narrow?
- Click the expand icon (maximize) in the header
- Panel will resize to 700px width
